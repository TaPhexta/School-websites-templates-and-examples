# Green Valley Primary School Website

## Project Overview
A multi-page website for Green Valley Primary School, striving for excellence.

## Structure
- **Pages**:
  - `index.html`: Home page
  - `academics.html`: Academic information
  - `sport.html`: Sports and physical education
  - `life.html`: Life at school (extracurriculars, pastoral)
  - `admissions.html`: Admissions information and process
  - `contact.html`: Contact information and form
- **Theme**: Black, White, and Red color scheme.
- **Components**: Shared Header, Footer, and Hero sections.
